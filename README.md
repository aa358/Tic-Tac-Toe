#❌Tic-Tac-Toe⭕

A browser-based Tic Tac Toe game built using HTML, CSS, and JavaScript. Players take turns clicking on cells to place Xs and Os, and the game detects wins or draws dynamically.

Skills Demonstrated:
- HTML
- CSS
- JavaScript
- DOM Manipulation
- Problem Solving
- Debugging.
